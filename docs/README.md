# Code Sync Project Documents

This folder contains academic-style project documentation for the `Code Sync` project.

Documents included:

- `CODE_SYNC_SRS.md`: Software Requirements Specification covering scope, functional requirements, non-functional requirements, architecture, interfaces, constraints, and acceptance criteria.
- `CODE_SYNC_SYNOPSIS.md`: Synopsis report covering abstract, problem statement, objectives, methodology, module description, feasibility, and future scope.

Project title used in these documents:

`Code Sync: A Real-Time Collaborative Coding and Communication Workspace`

These documents are based on the current implementation in this repository:

- React + Vite frontend
- Express + Socket.IO backend
- Judge0 integration for code execution
- Hugging Face or Pollinations integration for AI-assisted code generation
